Modular umfunktionierender Raum im bestehenden Raum

DESCRIPTION

c) Mindestanforderungen

Grundsätzlich kann das Prismae Raumkonzept in jedes bestehende Schulsystem integriert werden.

Voraussetzung ist, dass die einzelnen Räume mindestens neun und maximal fünfzehn Quadratmeter groß sind. Tren-nwände, ein Schienensystem und die oben beschriebene Technik können nachgerüstet werden. Bei einer Klassen-größe von achtzehn Schülern werden sechs Räume benötigt. Ausgegangen wird hierbei von einer Gruppengröße von maximal drei Schülern.
